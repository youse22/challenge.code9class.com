CodingChallenge Code9Class

Date de début: 24 mars 2024

Durée: 35 jours

Notre Objectif

Objectif de l'événement:

Offrir une opportunité à tous les passionnés de programmation de développer leurs compétences,

d'apprendre et de tester leurs connaissances dans cinq langages de programmation populaires.

Les langages à choisir

Langages au choix:

Python

PHP

Dart

JavaScript

C#

Test d'évaluation (20 mars 2024):

  • Avant le lancement officiel, un test d'évaluation sera organisé pour sélectionner les participants.
  • Le test comportera des exercices d'algorithmique, de logique et de déduction.
  • Les 60 meilleurs participants seront sélectionnés pour participer au Coding Challenge.

Frais de participation (après sélection):

  • Les participants sélectionnés devront payer un frais de 500 HTG, pour chaque langage choisi.

Déroulement de l'événement:

Phase d'apprentissage (24 mars - 22 avril 2024):

Les participants auront 30 jours pour maîtriser le langage de programmation de leur choix.

Ressources en ligne disponibles sur la plateforme Code9Class pour faciliter l'apprentissage.

Les participants peuvent poser des questions et interagir avec d'autres apprenants via le forum dédié.

Challenge de Codage (23 avril 2024):

Les participants qui réussissent le challenge sont invités à participer à la phase suivante.

Coding Battle en phase de poules (26 avril):

Les participants s'affrontent dans des duels de codage.

Les deux meilleurs de chaque poule avancent à la phase suivante.

Phase K.O (27avril 2024):

Les participants qualifiés s'affrontent dans des matchs à élimination directe jusqu'à la finale.

Finale (28 avril 2024):

Les finalistes compétitionnent pour les premiers prix.
